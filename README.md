# micli-cfg

## Installation (Temporary!)

``` bash
## with npx
$ npx github:renem2185/micli-cfg

## local install
$ sudo npm install -g git+https://github.com/renem2185/micli-cfg.git
```

## Usage

``` bash
$ micli-cfg --lang en

### interactive shell in English...
```

## つかいかた (Japanese)

``` bash
$ micli-cfg --lang jp

### 以下、日本語対話シェル
```
