# micli-cfg

## Installation (Temporary!)

``` bash
$ sudo npm install -g git+https://github.com/renem2185/micli-cfg.git
```

## Usage

``` bash
$ micli-cfg --lang en

(interactive shell in English...)
```

## つかいかた (Japanese)

``` bash
$ micli-cfg --lang jp

(以下、日本語対話シェル)
```

## Note

- `micli-tl` Timeline viewerとセットで使ってください
  - そのうち投稿コマンドも作るつもりです
